# מריה קוסמטיקס · Maria Cosmetics

A modern, mobile-first, RTL Hebrew website for **Maria Cosmetics** — a boutique beauty clinic in Ein Hayam, Hadera, Israel.

## Highlights

- 🌹 Warm rose/sand/gold theme tailored to a beauty brand
- 📱 Mobile-first, fully responsive
- 🕋 Hebrew RTL with semantic HTML and ARIA
- 🎨 3D hero scene (CSS perspective, floating petals, parallax tilt)
- 🌀 Scroll-reveal animations with IntersectionObserver
- ✨ Animated marquee, pointer-glow service cards, pulsing CTA
- 💬 Floating WhatsApp button + contact form that opens WhatsApp with prefilled details
- 🔗 Social links (Facebook, Instagram, TikTok)
- 🔍 SEO: meta tags, OpenGraph/Twitter cards, JSON-LD `BeautySalon`, sitemap, robots, manifest
- ♿ Accessibility: skip link, focus rings, `prefers-reduced-motion`, alt/labels, semantic landmarks
- ⚡ No build step — vanilla HTML/CSS/JS with Google Fonts via CDN

## Run locally

Just open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8000
# then visit http://localhost:8000
```

## Structure

